# MM_Accessibility_Layer

Multimodal Accessibility Layer (MAL) monorepo.

## Project Structure

- `frontend/` - React + Vite accessible UI prototype
- `backend/` - Node.js backend starter service
- `PRD.md` - Product Requirements Document

## Run Frontend

```bash
npm install
npm run dev:frontend
```

## Run Backend

```bash
npm run dev:backend
```
